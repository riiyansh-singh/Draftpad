# Draftpad

Draftpad is a draggable markdown-based Chrome extension built for students.

## Features

- Alt + N toggle shortcut
- Alt + S Smart Snip Selected Text
- Alt + Y Insert YouTube Timestamp
- Unlimited workspaces
- Auto-saving notes
- Minimal UI
- Keyboard-first workflow

## Installation

1. Clone repo
2. Open chrome://extensions
3. Enable Developer Mode
4. Click Load Unpacked
5. Select Draftpad folder

### Notes

## Installation & Usage

### 1. Download the Extension

- Click the green `Code` button on this repository
- Press `Download ZIP`
- Extract the ZIP file

---

### 2. Open Chrome Extensions

Open Chrome and go to:

chrome://extensions

Enable:

Developer Mode

(top-right corner)

---

### 3. Load the Extension

- Click `Load Unpacked`
- Select the extracted `Draftpad` folder

The extension is now installed.

---

### 4. How to Use

Open any website and press:

Alt + N

to open the Draftpad sidebar.
